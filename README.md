# LicenseKit Go SDK

Verify-side library for offline `.lkbundle` license files issued by
the LicenseKit backend.

> Pre-v1.0. API may change in minor releases until v1.0.0 ships.

## Install

```bash
go get github.com/LicenseKit/sdk_go
```

## Quick start

See [`examples/basic`](examples/basic) for a runnable example.
Full API reference in this README will land with v0.1.0.
